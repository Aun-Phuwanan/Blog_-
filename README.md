# FOBI คืออะไร
 FOBI คือหุ่นยนต์ที่ปฏิสัมพันธ์กับมนุษย์ได้ สามารถรับคำสั่งเสียงภาษาไทยกับมนุษย์ได้และยังสามารถเชื่อต่อกับอุปกรณ์ IoT โดยคุณสมบัติของ FOBI มีดังนี้  
 - พูดตอบโต้กับคนได้
 - หันหน้าตามคนได้
 - เล่นมุกตลกได้
 - ควบคุมอุปกรณ์ IoT ได้
